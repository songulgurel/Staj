Öğe seçici
 öğe adına göre HTML öğelerini seçer.
p {
  text-align: center;
  color: red;
}

Sınıf
Belirli bir sınıfa sahip öğeleri seçmek için, bir nokta ve bu noktadan sonra sınıfımıza vermek istediğimiz ad yazılır.
Kullanımı aşağıdaki örnekteki gibidir.
.class {
  text-align: center;
  color: red;
}

Aşağıdaki örnekteki gibi belirli HTML öğeleri bir sınıf tarafından da işaretlenebilir.
p.class {
  text-align: center;
  color: red;
}

Id
Id seçici, belirli bir öğeyi seçmek için bir HTML öğesinin kimlik niteliğini kullanır.
#para1 {
  text-align: center;
  color: red;
}

Evrensel seçici için '*' işareti kullanılır.
* {
  text-align: center;
  color: blue;
}
bu şekilde bütün html dosyasını etkileyen bir seçici tanımlanabilir.
